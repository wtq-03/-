# 四合院同人小说创作 - 验证清单

- [ ] 检查点1: 主角设定是否与参考作品无直接相似性
- [ ] 检查点2: 系统/金手指设定是否与参考作品有明显差异
- [ ] 检查点3: 场景描写是否符合六十年代北京的社会背景和生活细节
- [ ] 检查点4: 配角形象是否与原作有联系但具有独特性
- [ ] 检查点5: 故事情节是否与参考作品无直接抄袭
- [ ] 检查点6: 小说内容是否符合相关法律法规
- [ ] 检查点7: 主角名字是否通过搜索当前年份-1年的取名推荐获得
- [ ] 检查点8: 配角名字是否通过搜索当前年份-1年的取名推荐获得
- [ ] 检查点9: 开篇是否吸引人，符合黄金三章法则
- [ ] 检查点10: 情节发展是否合理，逻辑连贯
- [ ] 检查点11: 角色互动是否自然，符合各自性格
- [ ] 检查点12: 语言表达是否流畅，增强代入感和可读性
- [ ] 检查点13: 小说整体质量是否符合要求
- [ ] 检查点14: 版权风险评估是否完成，确保无侵权风险
- [ ] 检查点15: 最终审查是否完成，所有要求都已满足
