
# 长篇四合院小说规划

## 整体规划
- **小说名称**：《四合院：从1955开始的幸福生活》
- **总章节数**：300章+
- **时间跨度**：1955年-1985年（30年）
- **核心设定**：符合时代背景的系统，合理的情绪值收集，真实的四合院生活
- **写作风格**：贴近原作《四合院：我不爽，都别想好过》的爽文风格，同时保持合理性

## 时代背景要求
- 严格遵守1955-1985年的历史背景
- 物资凭票供应的真实体现
- 社会变革的合理描写
- 人物行为符合时代特征

## 系统简化与合理化
### 系统名称
情绪共鸣系统（简化版）

### 系统功能
1. **情绪值收集**：通过与四合院居民互动，收集情绪值
   - 正面情绪值：开心、感激、敬佩等
   - 负面情绪值：嫉妒、愤怒、不满等
2. **抽奖系统**：
   - 初级抽奖（100情绪值）：日常生活用品
   - 中级抽奖（500情绪值）：技能、知识、少量资金
   - 高级抽奖（2000情绪值）：特殊能力、大额资金
3. **系统空间**：10立方米（逐步扩大）
4. **新手礼包**：基础生活物资、初级机械维修技能

### 系统能力限制
- 不会出现超自然能力（如飞行、隐身、时间回溯等）
- 能力都是符合时代背景的实用技能
- 抽奖物品都是那个时代可以获得的

## 人物设定
### 主角：陈阳
- 现代穿越者，回到1955年
- 性格：善良、有正义感、智慧、不惹事但不怕事
- 技能：从系统获得的机械维修、医疗、烹饪等实用技能
- 事业线：从普通工人→技术骨干→车间主任→技术副厂长→厂长

### 主要女性角色
1. **秦淮茹**：温柔善良的寡妇，独自带三个孩子
2. **娄小娥**：许大茂前妻，善良有文化
3. **于海棠**：年轻寡妇，后来搬入四合院

### 主要男性角色
1. **傻柱**：食堂厨师，热心肠但有点冲动
2. **许大茂**：放映员，反派角色
3. **易忠海**：四合院一大爷，表面公正但有私心
4. **刘海忠**：四合院二大爷，官迷
5. **闫埠贵**：四合院三大爷，吝啬

### 其他角色
1. **贾张氏**：秦淮茹婆婆，自私刻薄
2. **贾东旭**：秦淮茹丈夫（已去世）
3. **棒梗、小当、小槐花**：秦淮茹的三个孩子
4. **轧钢厂的同事和领导**
5. 四合院的其他邻居

## 情绪值收集规则
- 每次互动收集的情绪值不超过100
- 负面情绪值可以收集，但要合理利用
- 情绪值的获取要自然，不能刻意制造冲突

2. 调整剧情节奏，不要进展太快
3. 简化系统，移除超自然能力
4. 让抽奖物品更符合时代背景
5. 调整感情线发展速度
6. 增加更多日常细节
7. 让人物行为更符合时代特征
